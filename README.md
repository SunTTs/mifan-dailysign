# 🍚 米饭APP每日签到脚本

一个基于青龙面板的[米饭APP](https://mifan.61.com/)自动签到脚本，使用Token验证方式，支持多账号签到。

## 🌟 功能特性

- ✅ 青龙面板完美兼容
- ✅ 自动每日签到米饭APP（0成本永续摩尔庄园超拉）
- ✅ 支持多账号签到（Token方式）
- ✅ 良好的日志记录和重试机制

## 🚀 使用方法

### 1. 青龙面板添加订阅

| 项目 | 值 |
| :--- | :--- |
| 名称 | `米饭APP签到` |
| 地址 | `https://github.com/SunTTs/mifan-dailysign.git` |
| 分支 | `master` |
| 定时 | `0 12 * * *`（每天上午12点执行） |

### 2. 配置环境变量

| 环境变量名 | 描述 | 示例值 |
| :--- | :---:| :--- |
| `MIFAN_TOKEN` | 米饭账号tokens | `token1;token2;token3` |

> 多账号使用英文分号 `;` 分隔，单账号不需要分号

### 3. 安装依赖

1. 在青龙面板左侧菜单中点击 `依赖管理` -> `创建依赖`
2. 选择 `nodejs` 类型并输入依赖名称 `axios`
3. 等待安装成功即可

### 4. 运行脚本

设置定时任务，建议时间：`0 9 * * *` （每天上午9点执行）

## 🔧 Token获取方法

1. 打开[米饭APP](https://mifan.61.com/)
2. 使用浏览器开发者工具
3. 登录账号后找到 login 返回的 `token` 字段
4. 复制 `token` 字段后面的字符串
5. 填入环境变量 `MIFAN_TOKEN`

> ⚠️ 请妥善保管你的 token，泄露会影响你的账号安全

## 🛠️ 常见问题

### 获取签到状态失败或签到失败
- 重新获取最新的Token
- 更新环境变量
